Spectral lines are drawn using Gaussian profiles with pixel size set to 0.1 nm where each line is calculated over 100 pixels.
Hover over spectrum to see the legend as well as inensity values at any pixel. Click and hold left mouse button to select a rectangle to zoom in. Right mouse button click to zoom out.
